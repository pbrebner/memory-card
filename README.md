# Memory Card Game

A simple memory card game with multiple levels where you need to try and select all characters once.

[Live Demo](https://pbrebner.github.io/memory-card/)

![My-Image](UI.png)

## Getting started

```
git clone https://github.com/pbrebner/memory-game.git
cd memory-game
npm install
npm start
```

-   React
-   Function Components
-   Hooks
